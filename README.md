# Student Management System

## Description
A beginner-friendly Java Swing GUI project to manage students.

## Features
- Add Student
- View Students
- Search Student by Roll Number
- Delete Student by Roll Number

## How to Run
1. Open the project in Eclipse.
2. Run MainScreen.java.
3. Use the buttons to add, view, search, or delete students.

## Future Improvements
- Save/load students to file.
- Improve GUI with JTable for better display.
